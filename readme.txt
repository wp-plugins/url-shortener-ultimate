###Url Shortener Ultimate 

Add as many url shortener ultimate links to your site as you like.  The Url Shortener Ultimate lets you create links and save them for when ever you are ready to use them.

== Installation ==


1. Upload `url shortener ultimate` to the `/wp-content/url-shortener-ultimate/` directory
1. Activate the url shortener ultimate through the 'Url Shortener Ultimate' menu in WordPress
1. Click **Url Shortener Ultimate** on the admin menu to enable and set your options.
